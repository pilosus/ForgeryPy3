internet Module¶

Generate random date-related data.

forgery_py.forgery.internet.user_name(with_num=False)[source]¶

Return a random user name.

Basically it’s lowercased result of first_name() with a number appended if with_num.

forgery_py.forgery.internet.top_level_domain()[source]¶

Return a random top-level domain name.

forgery_py.forgery.internet.domain_name()[source]¶

Return a random domain name.

Lowercased result of company_name() plus top_level_domain().

forgery_py.forgery.internet.email_address(user=None)[source]¶

Return random e-mail address in a hopefully imaginary domain.

If user is None user_name() will be used. Otherwise it will be lowercased and will have spaces replaced with _.

Domain name is created using domain_name().
