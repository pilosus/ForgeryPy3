russian_tax Module¶

Generate various random ID numbers for Russian banking/taxation system.

Some types of IDs may have different patterns for different types of legal personality: natural persons, sole contractors, local companies, international companies.

forgery_py.forgery.russian_tax.account_number()[source]¶

Return a random bank account number.

forgery_py.forgery.russian_tax.bik()[source]¶

Return a random bank identification number.

forgery_py.forgery.russian_tax.inn(type='legal')[source]¶

Return a random taxation ID number for either a person or a company.

Further information on the topic can be found in [1] (in russian). [1]: https://ru.wikipedia.org/wiki/Идентификационный_номер_налогоплательщика

forgery_py.forgery.russian_tax.legal_inn()[source]¶

Return a random taxation ID number for a company.

forgery_py.forgery.russian_tax.legal_ogrn()[source]¶

Return a random government registration ID for a company.

forgery_py.forgery.russian_tax.ogrn(type='legal')[source]¶

Return a random government registration ID for either a person or a company.

Further information on the topic can be found in [1] (in russian). [1]: https://ru.wikipedia.org/wiki/Основной_государственный_регистрационный_номер

forgery_py.forgery.russian_tax.person_inn()[source]¶

Return a random taxation ID number for a natural person.

forgery_py.forgery.russian_tax.person_ogrn()[source]¶

Return a random government registration ID for a person.
