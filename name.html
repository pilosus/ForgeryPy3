name Module¶

Generate random names and name-related strings.

forgery_py.forgery.name.first_name()[source]¶

Return a random male of female first name.

forgery_py.forgery.name.last_name()[source]¶

Return a random last name.

forgery_py.forgery.name.full_name()[source]¶

Return a random full name.

Equivalent of first_name() + ' ' + last_name().

forgery_py.forgery.name.male_first_name()[source]¶

Return a random male first name.

forgery_py.forgery.name.female_first_name()[source]¶

Return a random female first name.

forgery_py.forgery.name.company_name()[source]¶

Return a random company name.

forgery_py.forgery.name.job_title()[source]¶

Return a random job title.

forgery_py.forgery.name.job_title_suffix()[source]¶

Return a random job title suffix.

forgery_py.forgery.name.title()[source]¶

Return a random name title, e.g. Mr.

forgery_py.forgery.name.suffix()[source]¶

Return a random name suffix, e.g. Jr.

forgery_py.forgery.name.location()[source]¶

Return a random location name, e.g. MI6 Headquarters.

forgery_py.forgery.name.industry()[source]¶

Return a random industry name.
