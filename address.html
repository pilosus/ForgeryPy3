address Module¶

Generate forged address-related data.

forgery_py.forgery.address.street_name()[source]¶

Return a random street name.

forgery_py.forgery.address.street_number()[source]¶

Return a random street number.

forgery_py.forgery.address.street_suffix()[source]¶

Return a random street suffix.

forgery_py.forgery.address.street_address()[source]¶

Return a random street address.

Equivalent of street_number() + ' ' + street_name() + ' ' + street_suffix().

forgery_py.forgery.address.city()[source]¶

Return a random city name.

forgery_py.forgery.address.state()[source]¶

Return a random US state name.

forgery_py.forgery.address.state_abbrev()[source]¶

Return a random US abbreviated state name.

forgery_py.forgery.address.province()[source]¶

Return random Canadian province or territory.

forgery_py.forgery.address.province_abbrev()[source]¶

forgery_py.forgery.address.zip_code()[source]¶

Return a random ZIP code, either in ##### or #####-#### format.

forgery_py.forgery.address.phone()[source]¶

Return a random phone number in #-(###)###-#### format.

forgery_py.forgery.address.country()[source]¶

Return a random country name.
